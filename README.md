# Data Analysis - Cancelamento de Clientes 📊

## 📌 Sobre o Projeto
Este projeto realiza uma análise exploratória de dados com foco na identificação de padrões de cancelamento de clientes.

A base de dados foi tratada, limpa e analisada para identificar possíveis causas relacionadas à evasão de clientes.

## 🚀 Tecnologias Utilizadas
- Python
- Pandas
- Plotly
- CSV

## ⚙️ Etapas do Projeto

### 1️⃣ Importação e Tratamento dos Dados
- Remoção de colunas irrelevantes
- Tratamento de valores nulos
- Verificação de estrutura da base

### 2️⃣ Análise Exploratória
- Cálculo da quantidade de cancelamentos
- Percentual de clientes que cancelaram
- Visualização gráfica com histogramas

### 3️⃣ Identificação de Padrões
Análise das variáveis:
- Duração do contrato
- Número de ligações ao call center
- Dias de atraso

### 4️⃣ Aplicação de Filtros
Após análise dos dados, foram aplicados filtros estratégicos para simular ações de retenção e avaliar impacto na taxa de cancelamento.

## 🎯 Objetivo
Praticar análise exploratória de dados (EDA), manipulação com Pandas e geração de insights visuais utilizando Plotly.

## 💡 Aprendizados
- Limpeza e preparação de dados
- Uso de value_counts e normalize
- Criação de gráficos interativos
- Interpretação de padrões de comportamento
